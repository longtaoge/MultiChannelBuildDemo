# MultiChannelBuildDemo
Android 多渠道打包


本示例展示了AndroidStudio gradlew 进行多渠道打包的基本配置
学习本示例需具备基本 gradlew 基本知识。

本例完成了 对应用友盟渠道号的定制，同时在打包过程中对不同渠道的应用图片皮肤定制的基本功能
